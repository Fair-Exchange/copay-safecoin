{{'Advanced' | translate}} {{'Do not group wallets' | translate}}
If enabled, a list of wallets will not be grouped by coins in the Home tab.
{{'Use Unconfirmed Funds' | translate}}
If enabled, wallets will also try to spend unconfirmed funds.
{{'Use Bitcoin Cash Copay Style Addresses' | translate}}
If enabled, Bitcoin Cash addresses will be shown using Copay style address, and not the new cashaddr format.
{{'Recent Transaction Card' | translate}}
If enabled, the Recent Transactions card - a list of transactions occuring across all wallets - will appear in the Home tab.
